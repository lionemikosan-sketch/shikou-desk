# 思考デスク（Thinking Desk）

頭の中にあるメモ・TODO・悩み・アイデア・買い物候補・プロジェクト・保留事項を、
**すべてカードとして書き出し、整理し、行動に変える**ための、自分専用のオフライン思考整理アプリです。

- AI機能なし / ログインなし / 外部APIなし
- データはすべて端末内（IndexedDB）に保存され、**オフラインで動作**します
- PWA対応（インストールしてアプリのように使えます）

## 主な機能

| 画面 | 役割 |
| --- | --- |
| **ホーム** | 大きなクイック入力欄で「まず全部出す」。今日 / 受信箱 / 最近編集を一覧 |
| **受信箱** | 未整理カードを「今日へ / 今週へ / 保留へ / 完了へ / 編集 / 削除」で処理 |
| **カンバン** | 受信箱・今日・今週・作業中・保留・完了の6列。列の移動＋**列内の手動並べ替え**をドラッグ＆ドロップ（スマホは長押し）。種類で絞り込みも可能 |
| **マインドマップ** | カードをノードとして表示し、8種類の接続線（関係あり/原因/結果/必要/参考/アイデア/TODO/保留）でつなぐ |
| **テンプレート** | 悩み整理・アイデア整理・決断・プロジェクト整理・買い物比較の5種 |

そのほか：カード検索・タグ検索、**サブタスク（親子カード）**、**カード複製**、
JSONバックアップの書き出し／読み込み、全操作の自動保存。

## 技術構成

React + Vite + TypeScript / Tailwind CSS / Zustand / React Flow（@xyflow/react） /
IndexedDB（idb） / @dnd-kit / vite-plugin-pwa

## セットアップ

```bash
npm install      # 依存関係のインストール
npm run dev      # 開発サーバー（http://localhost:5173）
npm run build    # 本番ビルド（dist/）
npm run preview  # ビルド結果をプレビュー（PWA / オフラインの確認はこちら）
```

> オフライン動作・インストール（PWA）は本番ビルド（`npm run build` → `npm run preview`）で確認できます。
> 開発サーバーでは Service Worker を無効化しています。

## ディレクトリ構成

```
src/
  types/        型定義（Card / CardEdge など）
  db/           IndexedDB 永続化レイヤー
  store/        Zustand ストア（useStore: データ / useUI: 画面状態）
  lib/          純粋ロジック（検索・絞り込み・テンプレート・バックアップ・日付）
  components/   UI部品（共通・カンバン・マインドマップ・レイアウト）
  pages/        各画面
scripts/
  gen-icons.mjs PWAアイコン生成（依存ライブラリなし）
```

## データの保存とバックアップ

すべてのデータはブラウザの IndexedDB に保存されます。
機種変更や別ブラウザへの移行時は、左メニュー（スマホは右上）の
**「データ・バックアップ」→「JSONバックアップを書き出す」**でファイルを保存し、
移行先で**「読み込む」**を実行してください。
